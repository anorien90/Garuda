## Interaction Patterns

### Click
- Single click on node: Select node, show details, enable traversal

### Shift+Click
- Shift+click on nodes: Multi-select for path finding (max 2)

### Hover
- Hover over node: Show tooltip with node info
- Hover over edge: Show relationship type and metadata

### Keyboard
- Enter in search box: Trigger semantic search
- Escape: Close modals
- Tab: Navigate between controls
